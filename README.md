# rc-fun

> react helper functions

## Install

```bash
npm install rc-fun
```

## Usage

### bind

```javascript
import {bind} from 'rc-fun';

```

### setState

```javascript
import {setState} from 'rc-fun';

```

### switchState

```javascript
import {switchState} from 'rc-fun';

```

### ref

```javascript
import {ref} from 'rc-fun';

```
